复分解反应

复分解反应（双置换反应）是由两种化合物互相交换成分，生成另外两种化合物的反应。

复分解反应，有时又称双置换反应，是一种化学过程，涉及两种反应化学物质之间的键交换，从而产生具有相似或相同键合关系的产物。
此反应必发生在水溶液中，它是基本类型的化学反应之一。复分解都不是氧化还原反应（有些反应是复分解产物再发生氧化还原，而不是复分解的结果）。

维基百科复分解反应

记忆口诀：双交换，价不变
通式：
$\ce{AB + CD \xrightarrow{} AD + CB}$

反应发生条件：生成物中必须有

  • 沉淀 $\downarrow$
  • 气体 $\uparrow$
  • 水

酸、碱、盐 基本特点

酸

特点：水溶液中阳离子全部是 $\ce{H+}$
酸碱性：酸性 $\ce{pH < 7}$

碱

特点：水溶液中阴离子全部是 $\ce{OH-}$
酸碱性：碱性 $\ce{pH > 7}$

盐

特点：由金属离子（或铵根）和酸根离子构成
酸碱性：可酸性、可碱性、可中性

常见酸、碱、盐

常见酸

$\ce{HCl}$ 氯化氢水溶液——盐酸
$\ce{H2SO4}$ 硫酸
$\ce{HNO3}$ 硝酸
$\ce{CH3COOH}$ 乙酸——醋酸

常见碱

$\ce{NaOH}$ 氢氧化钠——烧碱、火碱、苛性钠
$\ce{Ca(OH)2}$ 氢氧化钙——熟石灰、消石灰
$\ce{NH3·H2O}$ 氨水

常见盐

$\ce{NaCl}$ 氯化钠——食盐
$\ce{Na2CO3}$ 碳酸钠——纯碱、苏打
$\ce{NaHCO3}$ 碳酸氢钠——小苏打
$\ce{CaCO3}$ 碳酸钙——大理石、石灰石主要成分
$\ce{CuSO4}$ 硫酸铜
$\ce{KMnO4}$ 高锰酸钾

必背化学方程式

酸 + 碱 → 盐 + 水

中和反应基础：
$\ce{HCl + NaOH \xlongequal{} NaCl + H2O}$

酸 + 盐 → 新盐 + 新酸

实验室制二氧化碳反应：
$\ce{2HCl + CaCO3 \xlongequal{} CaCl2 + H2O + CO2 ^}$

酸 + 金属氧化物 → 盐 + 水

盐酸除铁锈反应：
$\ce{6HCl + Fe2O3 \xlongequal{} 2FeCl3 + 3H2O}$

碱 + 非金属氧化物 → 盐 + 水

检验二氧化碳反应：
$\ce{Ca(OH)2 + CO2 \xlongequal{} CaCO3 v + H2O}$

碱 + 盐 → 新盐 + 新碱

蓝色沉淀生成反应：
$\ce{2NaOH + CuSO4 \xlongequal{} Cu(OH)2 v + Na2SO4}$

盐 + 盐 → 两种新盐

氯化银沉淀生成反应：
$\ce{AgNO3 + NaCl \xlongequal{} AgCl v + NaNO3}$

除杂提纯

盐酸中混硫酸： 加适量氯化钡，过滤
氢氧化钠中混碳酸钠： 加适量氢氧化钙，过滤
氧化钙中混碳酸钙： 高温煅烧
铜中混氧化铜： 加稀盐酸，过滤
二氧化碳中混氯化氢： 通过饱和碳酸氢钠溶液

✨ 易混易错＋细节

  • 水的氢离子浓度负对数，算出来刚好是7，就用它分界
  • pH 试纸、pH 计（精准测量）、紫色石蕊溶液、无色酚酞溶液都可以做酸碱指示剂
  • 指示剂变色是化学变化
  • 在复分解反应中，铁的化合价前后不变：可以是+2价也可以是+3价，只要没有置换、氧化还原，只是交换成分，铁价就不变
  • 生成盐和水的反应，不一定是中和反应
  • 不溶于水的碱，不能使指示剂变色
  • 硫酸钡、氯化银不溶于水也不溶于酸
  • 测 pH 时试纸润湿，只有中性溶液不受影响
  • 纯碱不是碱，是盐，溶液显碱性
